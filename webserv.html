Чем отличаются Web server и Application server?

Веб-сервер предназначен для обслуживания статических страниц, например HTML и CSS. Он принимает HTTP-запросы от клиентов, обычно веб-браузеров, и выдает им HTTP-ответы. Сервер-приложений — объект, который обрабатывает запросы, связанные с приложениями, для выполнения прикладных процессов (выборка данных, поиск данных, работа с терминалами).
